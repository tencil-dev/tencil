## Our Commitment

**Forever Free:** Tencil core tools (CLI, schema, adapters) will always 
be free and open-source under Apache 2.0.

**Sustainable:** We offer paid services (Tencil Cloud, Enterprise support) 
to ensure Tencil development is sustainable. These fund the free tools.

**No Lock-in:** If we ever shut down, your .tencil files remain usable. 
The schema is documented, the code is open.
